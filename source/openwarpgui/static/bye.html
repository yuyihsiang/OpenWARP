Software to help manage renewable wave energy

OpenWARP enables the simulation of hydrodynamics and device components. It contains a frequency domain boundary element method (BEM) module, OpenBEM. This software can be used to determine hydrodynamic coefficients and wave excitation forces. Learn More.
